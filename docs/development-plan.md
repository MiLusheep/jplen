# JPLen 开发计划

## 阶段一：项目基础搭建

### 任务 1.1：项目初始化
- 使用 Vite 创建 React + TypeScript 项目
- 配置 ESLint、Prettier
- 安装核心依赖（react-router-dom, zustand, framer-motion）

### 任务 1.2：设计系统搭建
- 创建 CSS 变量体系（色彩、间距、圆角、阴影）
- 全局样式重置与字体引入
- 动画关键帧定义
- 通用基础组件（Button, Card, ProgressBar, Modal）

### 任务 1.3：布局与导航
- 主布局组件（侧边栏 + 内容区）
- 响应式导航栏
- 页面路由配置
- 页面切换动画

## 阶段二：核心功能模块

### 任务 2.1：首页/仪表盘
- 学习进度概览
- 每日推荐内容卡片
- 快捷入口（继续学习、今日任务）
- 学习连续天数展示

### 任务 2.2：语法学习模块
- 语法级别选择器（N5→N1 分层）
- 语法课程列表（带完成状态标记）
- 语法详情页（解释 + 例句 + 练习）
- 语法练习与即时反馈
- 学习进度追踪

### 任务 2.3：阅读练习模块
- 阅读材料分类（新闻/故事/日常对话）
- 阅读器（支持词汇/语法注解）
- 难度自适应推荐
- 阅读理解练习
- 阅读历史记录

### 任务 2.4：跟读练习模块
- 课程选择与示例播放
- 麦克风录音功能
- 波形可视化
- 录音回放与对比
- 发音评分反馈

### 任务 2.5：词汇模块
- 词汇卡片（翻转动画）
- 词汇分类浏览
- 间隔重复复习
- 词汇收藏

## 阶段三：数据与体验优化

### 任务 3.1：学习数据
- 内置 N5 基础语法数据
- 内置基础词汇数据
- 内置初级阅读材料

### 任务 3.2：状态持久化
- 学习进度本地存储
- 录音文件 IndexedDB 存储
- 用户偏好设置持久化

### 任务 3.3：体验优化
- 加载状态与骨架屏
- 离线可用
- 微交互动画打磨
- 深色模式支持

## 当前版本目标（v0.1.0）

完成阶段一全部 + 阶段二的核心功能（语法学习、跟读练习、首页），确保可运行可交互。
